# Commercial Coordination

Use when a Productive or Moneybird action must connect to agency work in Notion.

Before calling either system, establish:

1. The concrete executable action.
2. Current authorization for that action.
3. The owning Task, Project, or Customer for writeback.

The coordination gate passes when all three are explicit. Otherwise return the exact missing boundary before calling either system through this coordination branch.

Use `productive-io` for hours evidence and time-entry operations. Use `moneybird` for estimates, invoices, recurring billing, and financial-document operations.

After an action, write the durable result, evidence, source link or identifier, and resulting agency-work state to the owning Notion record.
