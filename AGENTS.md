# AGENTS.md

## Communication
- Short, direct, useful. Cut filler.
- Keep technical substance; drop pleasantries, hedging, and repetition.
- Be critical and concrete; push back on weak plans.
- Prefer numbered lists for easy feedback.
- Avoid softeners like "maybe", "I think", "volgens mij", "misschien", "wellicht", "ik denk".
- Customer drafts: human, certain, only decision/scope/pricing/next action.

## Workspace Tools
- Notion: tasks, projects, customers, contacts, someday, goals, insights, meetings, work packages.
- Moneybird: quotes, invoices, recurring packages, pricing, finance-document cleanup.
- Protect private/customer/personal data and secrets.
- Ask before destructive, external, irreversible, or privacy-sensitive actions.
- Check live state when facts may have changed.

## Development Workflow
- Pull latest before starting work.
- Astro customer projects: work on `preview`; merge to `main` only when tested/publish-ready.
- Own Shopify customers: `preview` = preview theme, `main` = live theme.
- Teveo/Fayn Shopify via Small Giants: do not publish themes; push tested work to `main`; Sil handles theme publishing.
- Use separate branches only for real divergence, migrations, or major version splits.
